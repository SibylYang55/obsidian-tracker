## 主要功能

### 设置界面
这个插件主要用于打卡记录，插件设置界面可以自定义添加布尔、文字、数字类习惯。可以对习惯进行排序、修改等。
- 布尔类：按天/周设置频率，根据完成进度显示不同深浅颜色
- 文字类：点击弹出输入框，输入表情、符号、图标、文字
- 数字类：自定义单位和目标值，可以选择线图、条形图、组合图显示形式，根据目标完成情况显示不同深浅颜色
颜色可以到style.css文件里面修改RGB数值，和后面的透明度。

### 展示界面
由三部分组成，可以直接打卡并显示统计信息。
- Habit Check-in：周打卡部分，可切换上下周
- Monthly Overview：显示所有习惯当月的打卡情况，可切换上下月
- Habit Statistics：统计视图分为月视图和年视图，可分别按月和年切换
  月视图：只显示数字类习惯选择的数据图表
  年视图：布尔、文字类习惯显示按月热力图，数字类习惯按周显示选择的数据图表

## 安装

1.直接在obsidian第三方插件市场安装使用。
2.到该插件在GitHub的库中下载插件文件到本地，把文件放到obsidian库的plugins文件夹里。链接：
```
https://github.com/SibylYang55/tracker/tree/main
```
